About

This map diplays all current BAPL dealers and is filterable by Territory Manager.

Pin Colors

  • Johne Deere = green
  • CASE = red
  • AGCO = yellow
  • New Holland = blue
  • All other brands = purple

Search & Filtering

You can search geographic areas, not for specific dealers, in order to return a map depicting all the dealers within the selected search radius. It is also possible to select the checkboxes to view Dealers associated with a specific Territory Manager or Country.

Layers

Using the available layers, it is possible to overlay the Candaian or US territory boundries on the map. This is useful for quickly identifying geographic regions possible gaps in coverage of our Dealer Network.

When analyzing a territory for gaps and planning expansions, you can select the Dealer Prospects layer to depict the John Deere agricultural dealers in all US regions.
